# ZERO-Digital-Creative-Studio
One complete, production-grade, Awwwards-level file — WebGL shaders, Three.js, particle system, custom cursor, GSAP, the works.

## WebGL + Three.js Engine

Full-screen GLSL fragment shader with domain-warped 4-octave simplex noise — creates a living, breathing dark fluid atmosphere that reacts to your mouse in real-time
Dual-scene renderer (orthographic bg + perspective 3D) sharing one canvas
Triple-layered wireframe TorusKnot geometry (white + red-orange, different p/q ratios) slowly rotating with mouse parallax
1,500-point particle field drifting through 3D space

## Visual Design

Bebas Neue + Barlow Condensed + DM Mono — three typefaces with perfect hierarchy
Film grain animation (animated SVG noise, frame-stepped)
Animated rotating badge with conic-gradient ring
Clip-path cut-corner nav button
Scroll progress bar + auto-blur nav on scroll

## GSAP Animation System

Loader with percentage counter → hero reveal
Split-line text slide-in (translateY 115% → 0)
Stats counter that counts up when scrolled into view
Scroll-driven 3D card tilt effect (mouse-tracking rotationX/Y)
Staggered reveals on every section
Badge parallax that rotates and fades on scroll

### 7 Fully Styled Sections — nav, hero, marquee, about+stats, 2×2 work grid, scrolling type reel, 4-column process, contact CTA, footer.
